# Pycasso

Generate an ASCII image in python.

Made from a to z in 10 minutes.

## How to use


To install dependencies: 
```shell
pip install -r requirements.txt
```

To run pycasso :
```shell
python main.py <picture.png>
```

## Example

```shell
python main.py pepe.png
```

Input picture :

![pepe](pepe.png)
